# Ensemble Learning - Random Forest & Boosting

## Project Overview
Implementation of ensemble methods including Random Forest, AdaBoost, and Gradient Boosting for improved prediction accuracy.

## Techniques Covered
- Random Forest (Bagging)
- AdaBoost
- Gradient Boosting Machines (GBM)
- Voting Classifier

## Files Included
- ML Assignment Ensemble-Learning.ipynb

## Key Results
- Accuracy comparison between individual vs ensemble models
- Feature importance analysis
- ROC-AUC scores

## Requirements
scikit-learn, pandas, numpy, matplotlib, xgboost

## How to Run
1. Install required libraries
2. Open the Jupyter notebook
3. Run all cells sequentially
