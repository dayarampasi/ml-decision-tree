# Decision Tree Classification

## Project Overview
Implementation of Decision Tree algorithm for classification tasks from TopMentor ML curriculum.

## Key Features
- Decision tree classifier using scikit-learn
- Tree visualization and interpretation
- Handles both categorical and numerical data
- Pruning techniques to prevent overfitting

## Files Included
- ML Decision Tree.ipynb (Jupyter notebook)
- Dataset used for training

## Results
The decision tree model achieves good accuracy with interpretable rules including:
- Classification report
- Confusion matrix
- Feature importance scores

## How to Run
1. Install required libraries: pandas, numpy, scikit-learn, matplotlib
2. Open the Jupyter notebook
3. Run all cells
