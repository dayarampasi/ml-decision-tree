# Road Accident Analysis - Power BI Dashboard

## Dashboard Overview
This Power BI dashboard analyzes road accident data to identify patterns, high-risk areas, and key factors contributing to accidents.

## Key Features
- Accident severity analysis
- Location-based accident mapping
- Time-based trend analysis
- Vehicle type involvement

## How to View
1. Download the .pbix file
2. Open with Power BI Desktop (free)
3. Interact with filters and visuals
